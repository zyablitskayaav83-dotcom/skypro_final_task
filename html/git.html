Git

  • git status - показывает состояние рабочих папок и индекса, позволяя увидеть, какие изменения внесены и ожидают фиксации.
  • git checkout - перключение между различными ветвями или версионными снимками.
  • git clone - получение полного рабочего дерева из удаленного репозитория.
  • git push - отправляет локальные изменения в удаленный репозиторий.
  • git pull - получает изменения из удаленного репозитория и объединяет их с локальной версией.
